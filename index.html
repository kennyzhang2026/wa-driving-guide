📋 1. 驾照要求

好消息：可以！ 中国驾照持有者可以在西澳合法驾驶，但必须满足以下条件。

必须携带的证件

✅ 中国驾照原件

必须随身携带原件，电子照片/复印件不被接受

✅ 英文翻译件 / IDP

支付宝申请的国际驾照翻译认证件（小本本）可以使用，或者公证处出具的英文翻译件

✅ 护照

建议同时携带护照，以备身份核验

⚠️ 关于你的支付宝"国际驾照"小本本 中国大陆没有加入《日内瓦道路交通公约》，所以严格来说你拿到的不是"国际驾照（IDP）"，而是"驾照翻译认证件"。

在 WA 实际使用中：这个翻译件小本本 + 中国驾照原件 = 可以合法驾驶。
必须两本一起带！单独带小本本不行，必须同时出示原件。

关键规则

  • 驾照必须在有效期内 — 过期的不行
  • 只能驾驶你中国驾照上允许的车型 — C牌开小汽车没问题
  • 游客身份没有时间限制 — 整个访问期间都可以用
  • 永久居民需 3 个月内换领 WA 驾照 — 短期签证不受影响
  • 必须遵守所有 WA 交规 — 外国驾照没有豁免

中国驾照 vs 澳洲要求对比

项目中国澳洲 WA
驾照语言中文需要英文翻译件
必须随身携带是是（原件+翻译件）
电子驾照部分城市支持不接受，必须纸质原件
游客使用时限—整个访问期间
租车要求—驾照 + 翻译件 + 信用卡

🚗 2. 靠左行驶 — 最重要的一课

这是最大的不同！记住：KEEP LEFT！ 澳洲是左侧通行，与中国完全相反。每次上车前，先提醒自己"靠左"。

直观对比

🇨🇳 中国 — 靠右行驶

🚗

司机坐左边 → 方向盘在左边 → 车靠右走

🇦🇺 澳洲 — 靠左行驶

🚗

司机坐右边 → 方向盘在右边 → 车靠左走

关键差异一览表

项目中国 🇨🇳澳洲 WA 🇦🇺
行驶方向靠右靠左
司机位置左侧右侧
超车方向从左侧超从右侧超
快速车道最左侧最右侧
环岛方向逆时针顺时针
让行方向让左方来车让右方来车
红灯左转有时可右转永远不可左转
行人过街先看先看左先看右！

实用适应技巧

📌 仪表盘贴纸

在仪表盘贴一张"KEEP LEFT"提醒贴纸。很多租车公司会提供

📌 跟着前车

有车时跟着前车走，最不容易走错。没车时最容易出错

📌 转弯时最危险

左转是小弯（容易），右转是大弯（要跨对向车道），最容易走错方向

📌 雨刮/转向灯拨杆

澳洲车的转向灯和雨刮拨杆位置可能和中国相反，新手容易搞混

特别注意：红灯不能左转！ 在中国有些路口红灯可以右转，但在澳洲，红灯永远不能左转（除非有专门的绿色箭头灯指示可以左转）。这是很多中国司机被罚款的原因！

🚦 3. 限速规则

澳洲测速极其严格，容忍度仅约 1 km/h！ 不像国内可能有 10% 的缓冲，在 WA，标志上的数字就是绝对上限。

🔄 5. 环岛规则

核心规则：顺时针行驶，让右方来车！ 澳洲环岛是顺时针方向（与中国相反），进入环岛前必须让行已经在环岛内的车辆（从你右边来的车）。

三种通行方式

⬅ 左转（第一个出口）

打左转灯，走左车道，从左侧出口驶出

⬆ 直行

进入时不打灯，驶出前打左转灯

➡ 右转 / 掉头

打右转灯，走右车道，驶出前打左转灯

简单记忆法：
  • 进入环岛 → 看右，让右方来车
  • 在环岛里 → 顺时针走
  • 要出环岛 → 打左转灯

📖 6. 核心交规速查

红灯 = 必须停车
完全停住，等待绿灯。永远不能左转（除非有绿色箭头）
黄灯 = 必须停车
除非你已经太近刹不住。详见下方
绿灯 = 可以通行
但仍需让行已在校内的车辆和行人

黄灯详解（最容易犯错！）

常亮黄灯 = 能停就停，停不住才走
  • 黄灯亮起时，如果你离停止线还有足够距离安全刹车 → 必须停车
  • 黄灯亮起时，如果你已经非常接近停止线、急刹可能导致追尾 → 可以继续通过
  • 判断标准是"能否安全停车"，不是"能不能冲过去"
  • 闯黄灯罚款 $200 + 扣 2 分：摄像头拍到你有足够距离刹车却选择冲过去
闪烁黄灯 = 信号灯故障，谨慎通行
  • 跟常亮黄灯完全不同！不是"快点过"的意思
  • 表示信号灯关闭/故障，相当于没有红绿灯的路口
  • 可以通行，但必须按让行规则（让右方来车）小心通过
中国司机最常犯的错误： 国内习惯"黄灯加速冲过去"，这在澳洲会被罚款！看到黄灯的第一反应应该是刹车，不是加速。

箭头灯

  • 绿色箭头：可以按箭头方向行驶（此时主灯可能是红灯，但箭头方向可以走）
  • 黄色箭头：同常亮黄灯规则 — 能停就停
  • 红色箭头：箭头方向禁止通行（即使主灯是绿灯）
驾照类型血液酒精限制
正式驾照（Full License）0.05%
学习驾照 / 临时驾照0.00%（完全禁酒）

建议：开车就不喝酒！ 但了解一下"标准饮酒量"概念还是有用的。

什么是"一杯标准饮酒量"（Standard Drink）？

澳洲定义：含 10 克纯酒精 = 1 个标准饮酒量。不同酒的实际量差别很大：

酒类酒精度= 1 标准量的实际容量
普通啤酒4-5%约 285-375ml（一听/一罐）
精酿/IPA6-7%约 200-250ml（半听都不到）
红酒/白酒12-13%约 100ml（普通酒杯的 2/3）
烈酒（威士忌等）40%约 30ml（一个 shot）
鸡尾酒不定一杯可能 = 1-3 个标准量
实用技巧：
  • 澳洲所有酒瓶/酒罐上必须标注标准饮酒量数（如 "1.4 Standard Drinks"），买的时候看标签
  • 一罐 375ml 普通啤酒 ≈ 1.4-1.5 个标准量（不是"一杯"！）
  • 一杯餐厅葡萄酒（150ml）≈ 1.5 个标准量
  • 体型小的人可能 1 个标准量就到 0.05%，体型大的可能 2 个

WA 警察会随机路检吹气，拒绝检测是违法的。最安全的选择：开车日完全不饮酒。

道路标线规则：
  • 单实线 → 不能跨越超车
  • 双实线 → 绝对不能跨越
  • 虚线 → 安全时可以跨越
  • 实线+虚线 → 只有虚线在你这一侧时才能跨越
  • 例外：为了超越自行车，可以跨越实线（保持最小距离）
  • 限速 ≤60km/h 的道路：最少保持 1 米距离
  • 限速 >60km/h 的道路：最少保持 1.5 米距离
  • 必要时可以跨越中心线超自行车（安全情况下）
SLOMO = Slow Down, Move Over
  • 看到停在路边的紧急车辆闪灯（警车/救护/消防）→ 减速到 40km/h
  • 听到/看到迎面而来的紧急车辆 → 靠左让行

🦘 8. 野生动物 — 独特的澳洲挑战

每年超过 7000 起车辆撞袋鼠事故！ 这是澳洲驾驶最独特的风险。如果自驾去郊区或西澳内陆，必须了解。

高风险时段和区域

🌅 黎明和黄昏

袋鼠最活跃的时段。日出前 1 小时到日出后 1 小时，日落前 1 小时到日落后 1 小时

❄️ 冬季

袋鼠在路边找食物，冬季活动更频繁

🌾 郊区和乡村公路

城市里基本不会遇到，但开出珀斯 30 分钟后就有可能

🛣 长直路段

动物容易被车灯惊吓而跳向车辆，夜间行车尤其注意

遇到动物怎么办？

做法✅ 正确❌ 错误
发现动物减速 + 鸣笛高速闪避/急打方向盘
无法避开刹车 + 正面撞（更安全）急转弯（可能翻车/撞树）
撞到之后靠边停车 + 打电话靠近受伤袋鼠
如果你撞到了野生动物：
  1. 安全靠边停车
  2. 拨打 Wildcare 热线：9474 9055
  3. 不要靠近受伤的袋鼠 — 后腿踢击力量极大，非常危险
  4. 如果有人员受伤，拨打 000

🅿️ 9. 停车规则

标志含义违规后果
No Stopping
（或路边实线黄线）		 绝对不能停车，上下客也不行 罚款
No Parking 可临时停车 2 分钟上下客，司机不能离开车 3 米以外 罚款

珀斯停车小贴士：
  • 购物中心通常免费停车 2-3 小时
  • 路边停车看清楚标志上的时间段（如"2P 8am-5:30pm"意味着过了 5:30pm 就不限时了）
  • 很多停车场支持 APP 缴费（如 CellOPark, EasyPark）
  • 停车方向：必须与车流同方向停车（靠左停，车头朝前）

交通事故处理步骤

  1. 确保安全 — 开双闪，移至安全位置（如果可以移动）
  2. 检查人员 — 是否有人受伤
  3. 拨打 000 — 如果有人受伤或车辆无法移动
  4. 交换信息 — 对方姓名、地址、驾照号码、车辆注册号、保险公司
  5. 拍照留证 — 拍摄现场、车辆损坏、对方车牌
  6. 报告保险 — 租车的话联系租车公司
推荐 APP：
  • St John First Responder — 呼叫救护车时自动发送 GPS 定位
  • Main Roads WA Travel Map — 实时路况
  • emergency.wa.gov.au — 紧急警报（山火等）
